# 부록

## 부록 A. 입력 데이터 구성

본 연구의 시범적용 입력은 `data/input_assets.xlsx`와 `data/cnaim_params.xlsx`에 저장되어 있다. 입력 데이터는 6개 주요 배전설비, 총 21,893기 설비로 구성되며, 2026~2030년 5개년 계획기간을 전제로 한다.

| 항목 | 값 또는 파일 |
|---|---|
| 입력 자산 데이터 | `data/input_assets.xlsx` |
| CNAIM 파라미터 | `data/cnaim_params.xlsx` |
| 설비 수 | 21,893기 |
| 설비유형 수 | 6종 |
| 계획기간 | 2026~2030년 |
| 설비별 최대 교체횟수 | 5년 동안 최대 1회 |

## 부록 B. 설비유형별 정전시간 가정

| 설비유형 | 정전시간(분) | 구분 |
|---|---:|---|
| 주상변압기 | 95 | 사용자 지정 |
| 지상변압기 | 100 | 사용자 지정 |
| 가공개폐기 | 78 | 사용자 지정 |
| 지중개폐기 | 100 | 사용자 지정 |
| 가공배전선로 | 60 | 사용자 지정 |

정전시간은 실제 고장복구 이력, OMS/DMS 자료, 피더별 고객수 자료가 확보되면 대체 또는 보정해야 한다.

## 부록 C. 5개년 PoF·Risk·투자효과 산출 파일

5개년 PoF·Risk·투자효과 산출물은 `data/pof_5yr_output.xlsx`이다.

| 주요 컬럼 | 내용 |
|---|---|
| `pof_2026`~`pof_2030` | 설비별 연도별 PoF |
| `risk_2026_kkrw`~`risk_2030_kkrw` | 설비별 연도별 Risk |
| `saidi_2026_min`~`saidi_2030_min` | 설비별 연도별 SAIDI 기여도 |
| `replacement_cost_2026_kkrw`~`replacement_cost_2030_kkrw` | 연도별 교체비용 현재가치 |
| `risk_reduction_2026_kkrw`~`risk_reduction_2030_kkrw` | 설계수명 기준 Risk 저감량 |
| `investment_value_2026_kkrw`~`investment_value_2030_kkrw` | Risk 저감량 - 투자비용 |
| `investment_efficiency_2026`~`investment_efficiency_2030` | Risk 저감량 / 투자비용 |
| `candidate_top30_current` | 2026년 기준 Risk 또는 투자가치 상위 30% 후보군 |

## 부록 D. 단일지표 최적화 결과 파일

최적화 결과는 실행 시점별 폴더 `outputs/simulation_chapter_v2_matlab_YYYYMMDD_HHMMSS/`에 저장한다. 주요 결과 파일은 `simulation_chapter_v2_results.xlsx`이며, 단계별 중간 결과는 `checkpoints` 폴더에 별도로 저장한다.

| 시트 | 내용 |
|---|---|
| `constraints` | 연도별 예산·물량 제약 |
| `selected_assets` | 기법별 선택 설비 상세 |
| `value_phase` | 설비유형별 Risk 그리디, 투자가치 그리디, 투자가치 ILP, 투자가치 GA 결과 |
| `pi_phase` | 설비유형별 PI 그리디, PI ILP, PI GA 결과 |
| `integrated_phase` | 사후 가중 방식과 사전 PI_통합 방식의 통합설비 최적화 결과 |
| `comparison_total` | 주요 기법별 합산 비교표 |
| `comparison_by_type` | 설비유형별 비교표 |
| `baseline_compare` | Risk 그리디 또는 투자가치 ILP 대비 증감률 |

## 부록 E. 구현 파일 요약

| 파일 | 역할 |
|---|---|
| `code/cnaim_clean_pipeline.py` | 입력 데이터, CNAIM 파라미터, 5개년 PoF·Risk·투자효과 산출 |
| `code/local_pi_matlab.m` | AHP 기반 Local PI와 Fuzzy 보정 Local PI 산출 |
| `code/simulation_chapter_v2_matlab.m` | 설비유형별 단일지표 최적화, PI 최적화, 통합설비 최적화를 일괄 실행 |
| `code/postprocess_optimization_effect_tables.py` | 필요한 경우 최적화 결과표와 베이스라인 비교표를 후처리 |
