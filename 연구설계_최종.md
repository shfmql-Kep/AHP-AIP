## 1. 연구 목적 및 신규성

### 1.1 연구 목적

기존 AIP(Asset Investment Planning) 체계는 CNAIM 화폐화 리스크(PoF×CoF)로 **개별 설비**의 교체 우선순위를 산출한 뒤 상위 N개를 순차적으로 선택(Greedy)하는 방식이다. 이 방식은 다음 두 가지 구조적 문제를 내포한다.

1. **단일 지표 편중**: 리스크 외 신뢰도(SAIDI), 현장제약, 사회·환경 영향이 의사결정에서 배제됨
2. **자산군별 독립 처리**: 예산을 자산군(변압기·개폐기·선로 등)별로 사전에 분할 배정한 뒤 각 자산군 내에서만 순위를 결정함 → 자산군 간 예산 비율 자체가 최적화되지 않음

본 연구는 이 두 문제를 동시에 해결하기 위해 **다기준 점수(AHP/Fuzzy AHP) + 0-1 ILP 최적화**를 결합하여, 이종 자산군을 단일 최적화 공간에서 동시에 고려하는 **전략적 설비 투자 포트폴리오 도출 체계**를 개발한다.

---

### 1.2 신규성 (3종)

| # | 신규성 | 내용 |
|---|--------|------|
| ① | 국내 최초 다기준 AIP | CNAIM 화폐화 리스크 + 다차원 정량지표 통합 체계 (RISS·KCI·DBpia 선행연구 부재 입증) |
| ② | **이종 자산군 동시 포트폴리오 최적화** | 자산군별 예산 분리 관행 탈피 → 전체 예산 하에서 이종 설비 동시 최적화 (핵심 차별점) |
| ③ | 전문가 현장의견 3계층 구조 통합 | 기준 가중치 / 현장 데이터 정량화 / 최적화 제약식의 3단계로 현장의견을 수학적으로 반영 |

---

### 1.3 기존 방법의 한계 (본 연구 출발점)

#### (A) 단일 지표 편중
- CNAIM Monetised Risk만 사용 → 동일 Risk 자산 간 우선순위 모호
- 현장 경험·운영제약이 의사결정에서 구조적으로 배제 → 본사-현장 투자집행 GAP

#### (B) 자산군별 예산 독립 배분 관행 (연구 공백의 핵심 근거)

국내외 주요 배전망 사업자는 투자 예산을 자산군별로 독립 배정하는 관행을 유지하고 있음.

| 출처 | 내용 |
|------|------|
| **ENWL RIIO-ED1** (영국) | LV·HV Plant 21%, EHV Plant 17%, 철탑 13% 등 10개 자산군 독립 예산 배정 |
| **US EIA (2023)** | 가공선 34%, 지중선 23%, 변압기 15%, 변전소 12% — 자산군별 실적 보고 |
| **AER Repex 모델** (호주) | 공식 6개 자산군(전주·가공선·지중케이블·인입선·변압기·개폐기) 프레임워크 채택 |

→ 세 국가 모두 자산군 간 예산 비율을 수학적으로 최적화하는 사례 없음
→ 본 연구: 이종 자산군을 단일 최적화 공간에서 동시 처리 → **실질적 연구 공백**

## 2. 합성 데이터 순환논리 방어 전략

> **포지셔닝**: 본 논문의 기여 단위는 "PoF 예측 정확도"가 아닌 **"의사결정 체계(framework)"**다.

| 약점 프레임 | 강점 재포지셔닝 |
|---|---|
| CNAIM 데이터를 CNAIM으로 검증 → 순환 | CNAIM Risk는 4개 기준 중 1개. Baseline(단일 Risk 정렬)과 Proposed(다기준+현장의견)는 입력을 공유하되 **의사결정 로직이 다름** → 동일 입력 하에서의 로직 비교 |
| 합성 데이터 = 한계 | 합성 데이터 = **통제 실험**: ① 동일 Risk 자산 인위 생성(모호성 재현) ② 파라미터 전구간 커버 ③ 완전한 재현성 |

**보완 장치**
- 합성 데이터 생성 시 국내 배전설비 공개통계(평균 사용연수, 자산군 구성비)를 파라미터 분포에 반영 → 현실 정합성 확보
- 외적 타당성 한계는 정직하게 명시, "실계통 적용은 향후 연구"로 분리

## 3. 현장의견 3계층 통합 메커니즘

```
[계층 1] 기준 가중치            ← 전문가 AHP/Fuzzy AHP 쌍대비교 (5~9명)
[계층 2] '운영제약·실행가능성' 기준 ← 현장 데이터/의견 정량화
          (접근성, 정전 가능 시간대, 인력 가용성, 민원)
[계층 3] 최적화 제약식           ← 현장 실행 제약 직접 반영
          (동시 정전 한계, 피더 단위 작업, 의무교체 기한)
```

## 4. 방법론 체계 (확정)

```
[데이터]
  CNAIM v2.1 R 패키지 → 다종 배전설비 합성 데이터셋 생성
  (주상·지상변압기, 가공·지중개폐기, 배전선로)
  + 국내 공개통계로 파라미터 분포 보정

[기준 확정]
  경량 Delphi 1~2라운드 (소수 전문가) → 평가기준·계층 확정
  * Delphi는 독립 방법론이 아닌 기준 확정 도구로만 운영

[Stage 1: 가중치 산출 — 2종 비교]
  연구질문: "전문가 판단의 모호성·불확실성 반영이 투자 포트폴리오 결과를 유의하게 바꾸는가?"

  ① Classical AHP  : 쌍대비교 기반, 전문가 판단 집계 (기준선)
  ② Fuzzy AHP      : TFN 기반, 전문가 판단의 모호성·불확실성 보완

  → 두 방법 간 가중치·순위 비교 → 도메인 적합 방법 추천
  → 자산별 종합점수 Sⱼ 산출 (통합 비교 가능 점수)

[Stage 2: 최적화 — 2종 비교]
  연구질문: "정확 해법(ILP) 대비 발견적 해법(GA)의 실용성은?"

  ① 0-1 ILP (정수선형계획법)
       max Σ Sⱼ · xⱼ
       s.t. Σ Cⱼ · xⱼ ≤ B         (예산 제약)
            Σ Tⱼ · xⱼ ≤ T_max      (총 정전시간 제약)
            Σ Wⱼ · xⱼ ≤ W_max      (인력 제약)
            xₘ = 1  ∀m ∈ M         (의무교체 설비)
            현장 실행제약(계층3)
            xⱼ ∈ {0, 1}

  ② GA (유전 알고리즘) : 대규모 문제 대응 발견적 탐색, 동일 목적함수·제약

  → 해의 질(목적함수 값), 계산 시간, 확장성 비교

[비교 분석 — 3층 구조]
  층 1. Baseline vs Proposed
        · Baseline : CNAIM Risk 내림차순 Greedy 선택
        · Proposed : AHP/Fuzzy AHP 다기준 점수 + ILP 최적화
        · 지표     : 총 리스크 저감량, SAIDI 개선, 예산 활용률

  층 2. 통합 포트폴리오 vs 자산군별 개별 배분 (관행 vs 제안)
        · 자산군별 : 자산군마다 예산 독립 배정 후 각 군 내 Greedy 선택
        · 통합     : 전체 예산 하에서 이종 자산군 동시 ILP 최적화
        · 지표     : 포트폴리오 다양성, 총 리스크 저감량, 경계 자산 교체 여부

  층 3. 최적화 기법 비교 (ILP vs GA)
        · 소규모(N≤50): 최적해 일치 여부, 계산 시간
        · 대규모(N>200): Gap%, 실행 가능성, 실용성

[검증 보완]
  - Spearman 순위상관 (Classical AHP vs Fuzzy AHP 순위 안정성)
  - Top-N 변동 분석 (상위 20% 포트폴리오 구성 변화)
  - 민감도: 가중치 ±20%, 예산 ±30%
```

### 4.1 평가기준 4개 (확정)

| # | 기준 | 데이터 출처 | 정규화 | 성격 |
|---|------|------------|--------|------|
| C1 | CNAIM Monetised Risk (PoF × CoF) | CNAIM v2.1 R 패키지 | Min-Max | 객관 (투자가치) |
| C2 | SAIDI 기여도 | 합성 (고객수·복구시간) | Min-Max | 공급신뢰도 |
| C3 | 사회·환경 영향 | 합성 (위치·민원지수) | Min-Max | 정성 |
| C4 | 운영제약·현장성 | 합성 (접근성·정전가능시간·인력) | Min-Max | **현장의견(계층2)** |

### 4.2 수치 우월성 입증 (핵심 논리)

동일 예산 하에서 ILP 기반 통합 최적화가 Greedy 대비 높은 리스크 저감량을 달성함을 수치 예시로 입증:

```
설비  Risk  비용  | Greedy(단일Risk) | ILP(다기준)
  A   100    60   |     선택 ✓       |    제외
  B    80    40   |     제외          |    선택 ✓
  C    70    70   |     제외          |    제외
  D    60    40   |     선택 ✓       |    선택 ✓
  E    55    20   |     예산 초과     |    선택 ✓

예산 100 기준:
  Greedy: A(60)+D(40)=예산100, Risk저감 160
  ILP   : B(40)+D(40)+E(20)=예산100, Risk저감 195  (+21.9%)
```

→ 동일 예산에서 ILP 포트폴리오가 더 높은 리스크 저감 달성 = 수치 우월성 입증
