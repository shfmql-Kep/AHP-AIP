## 1. 연구 목적 및 신규성

### 1.1 연구 목적
나이·상태·Risk·투자가치를 **단일 지표로 고려할 때 발생하는 편중**과 **현장 의견 수렴 불가** 문제를 해결하기 위해,
정량 지표와 현장 전문가 의견을 종합한 **투자가치 기반 전력설비 투자 의사결정 체계**를 개발한다.

### 1.2 이중 신규성
- **신규성 ①**: 투자가치(Monetised Risk) + 다차원 정량지표 통합 — **국내 최초**
  (RISS·KCI·DBpia 검색으로 국내 선행연구 부재를 III장에서 명시적으로 입증)
- **신규성 ②**: 정량지표에 **현장 전문가 의견을 ANP로 구조적 통합** — 단일 Risk의 "현장의견 수렴 불가" 한계 해결

### 1.3 기존 방법의 한계 (본 연구 출발점)
1. Risk Matrix / 진단결과 / Risk 기반 투자가치 정량화 → 모두 **단일 차원 편중**
2. 동일 Monetised Risk 자산 간 우선순위 모호
3. 현장 경험·운영제약이 의사결정에서 배제 → 본사–현장 투자집행 GAP

## 2. 합성 데이터 순환논리 방어 전략 (최우선 리스크 대응)

> **재포지셔닝**: 본 논문의 기여 단위는 "PoF 예측 정확도"가 아니라 **"의사결정 체계(framework)"**다.

| 약점 프레임 | 강점 재포지셔닝 |
|---|---|
| CNAIM 데이터를 CNAIM으로 검증 → 순환 | CNAIM Risk는 **4개 기준 중 1개**. 베이스라인(단일 Risk 정렬)과 제안모델(다기준+현장의견)은 입력은 공유하되 **의사결정 로직이 다름** → "동일 입력에 대한 로직 비교" |
| 합성 데이터 = 한계 | 합성 데이터 = **통제 실험**. ①동일 Risk 자산 인위 생성(모호성 재현) ②파라미터 전구간 커버 ③완전한 재현성 |

**보완 장치**
- 합성 데이터 생성 시 국내 배전설비 공개통계(평균 사용연수, 자산군 구성비)를 파라미터 분포에 반영 → 현실 정합성 확보
- 외적 타당성 한계는 정직하게 명시, "실계통 적용은 향후 연구"로 분리

## 3. 현장의견 3계층 통합 메커니즘 (신규성의 실체)

```
[계층 1] 기준 가중치           ← 전문가 ANP 쌍대비교 (5~9명)
[계층 2] '운영제약·실행가능성' 기준 ← 현장 데이터/의견 정량화
          (접근성, 정전 가능 시간대, 인력 가용성, 민원)
[계층 3] 최적화 제약식          ← 현장 실행 제약 직접 반영
          (동시 정전 한계, 피더 단위 작업)
```

## 4. 방법론 체계 (확정)

```
[데이터] CNAIM v2.1 → 다종 배전설비 합성 (주상·지상변압기, 가공·지중개폐기, 선로)
         + 국내 공개통계로 파라미터 보정

[기준 확정] 경량 Delphi 1~2라운드 (소수 전문가) → 평가기준·계층 확정

[Stage 1: 가중치 산출 — 비교+추천, ANP는 의존성 검정 도구]
   연구질문: "투자기준 간 상호의존성이 우선순위 결과를 유의하게 바꾸는가?"
   ① Classical AHP : 독립 가정 기준선
   ② Fuzzy AHP     : 전문가 판단 모호성 보완
   ③ ANP           : 기준 간 의존성 반영 → 의존성 효과 검정
   → 검정 결과로 도메인 적합 방법을 "데이터로" 추천
      · 순위 변화 작음 → AHP/Fuzzy AHP 채택 정당화 (단순성 우위)
      · 순위 변화 큼   → ANP 채택 정당화 (의존성 필수)
   → 자산별 종합점수 Sⱼ

[Stage 2: 최적화]
   0-1 ILP : max Σ Sⱼ·xⱼ
             s.t. 예산 · 정전시간 · 인력 · 의무교체 · 현장 실행제약(계층3)

[검증]
   - Baseline(CNAIM 단일 Risk) vs 제안모델
   - 통합 단일 포트폴리오 vs 자산군별 개별 랭킹 → 둘 다 수행 후 비교
   - Spearman 순위상관, Top-N 변동, 예산활용률
   - 민감도: 가중치 ±20%, 예산 ±30%
```

### 4.1 평가기준 후보 (4개)
| 기준 | 데이터 출처 | 정규화 | 성격 |
|---|---|---|---|
| CNAIM Monetised Risk | R 패키지 | Min-Max | 객관(투자가치) |
| SAIDI 기여도 | 합성(고객수·복구시간) | Min-Max | 공급신뢰도 |
| 사회·환경 영향 | 합성(위치·민원지수) | Min-Max | 정성 |
| 운영제약·현장성 | 합성(접근성·정전가능시간·인력) | Min-Max | **현장의견(계층2)** |

## 6. 남은 실행 과제 (지도교수 협의 권장 순)

1. **국내 연구 부재 입증** — RISS·KCI·DBpia 검색 기록 확보 (신규성 핵심 근거)

5. **자산군 간 비교가능성 정당화** — 통합 포트폴리오 정규화 논리
