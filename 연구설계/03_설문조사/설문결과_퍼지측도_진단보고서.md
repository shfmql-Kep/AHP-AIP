## 2. 현재 설문 구조상 추가 발견 문제

현재 설문지는 3대 기준을 다음과 같이 구성한다.

| 대기준 | 하위지표 |
|---|---|
| 경제성 | 투자가치, 투자효율 |
| 신뢰도 | SAIDI 저감, 고장예측 저감 |
| 안전·환경 | 안전 영향, 환경 영향 |

따라서 기존 논문에서 사용하던 `경제성·신뢰도·Risk` 및 `재무 Risk·안전 Risk·환경 Risk` 구조와 다르다. 특히 재무 Risk가 별도 하위지표로 존재하지 않고, 대기준도 `Risk`가 아니라 `안전·환경`으로 바뀌어 있다.

검토 결과, 이 구조는 최종 연구모형으로 채택한다. 재무 Risk는 투자가치와 투자효율에 이미 반영되므로 MCDM 하위기준에 다시 포함하면 중복이 발생한다. 따라서 논문 본문은 현재 설문 구조에 맞춰 다음 기준으로 통일한다.

1. 경제성: 투자가치, 투자효율
2. 신뢰도: SAIDI 저감, 고장예측 저감
3. 안전·환경: 안전 영향 저감, 환경 영향 저감

## 4. 퍼지측도 점수와 λ 재계산

현재 설문은 퍼지측도 단일평가 점수를 `Wa = score / 6`으로 변환한다.

| 하위지표 | 평균점수 | Wa | 최소 | 최대 |
|---|---:|---:|---:|---:|
| 투자가치 | 4.7500 | 0.791667 | 4 | 6 |
| 투자효율 | 3.9500 | 0.658333 | 3 | 6 |
| SAIDI저감 | 4.9500 | 0.825000 | 4 | 6 |
| 고장예측저감 | 3.1500 | 0.525000 | 1 | 5 |
| 안전영향 | 4.9500 | 0.825000 | 3 | 6 |
| 환경영향 | 3.0500 | 0.508333 | 2 | 5 |
| 합계 |  | 4.133333 |  |  |

Sugeno λ-퍼지측도 조건은 다음과 같다.

```text
Π(1 + λ·Wa_i) = 1 + λ
```

재계산 결과:

```text
ΣWa = 4.133333
λ = -0.99948636
```

따라서 사용자가 지적한 `λ ≈ -1` 문제는 실제 응답자료에서 그대로 발생한다.

## 5. 문제 1: λ ≈ -1 포화 진단

### 5.1 판단

이 문제는 실제로 존재한다.

`λ=-0.999486`은 이론적 하한인 -1에 거의 붙어 있으므로, 현재 퍼지측도는 온건한 중복 보정이라기보다 거의 완전 대체적 측도에 가깝다.

### 5.2 원인

원인은 응답자가 잘못했다기보다 척도 설계에 있다.

전문가가 6개 지표를 대부분 “중요” 이상으로 평가하는 것은 자연스럽다. 그런데 이를 그대로 `score/6`으로 변환하면 각 `Wa`가 0.5~0.825 수준이 되고, 6개를 합산하면 4.133까지 올라간다. 이 상태에서는 Sugeno λ가 구조적으로 -1 근처로 밀린다.

### 5.3 결과 영향

현재 Shapley 보정 결과는 다음과 같다.

| 하위지표 | AHP Wr | Shapley φ | 차이 |
|---|---:|---:|---:|
| 투자가치 | 0.225110 | 0.195391 | -0.029719 |
| 투자효율 | 0.129016 | 0.156316 | +0.027300 |
| SAIDI저감 | 0.203753 | 0.205806 | +0.002054 |
| 고장예측저감 | 0.057898 | 0.120486 | +0.062588 |
| 안전영향 | 0.263331 | 0.205806 | -0.057525 |
| 환경영향 | 0.120893 | 0.116194 | -0.004699 |
| 합계 | 1.000000 | 1.000000 |  |

표면적으로는 AHP와 Fuzzy-AHP가 차이를 보인다. 그러나 이 차이가 전문가가 의도한 “중복성 반영”인지, `λ≈-1` 포화가 만든 기계적 평탄화인지 구분하기 어렵다.

### 5.4 민감도 확인

`Wa`의 상대적 비율은 유지하고 합계만 조정했을 때 λ는 다음과 같다.

| 목표 ΣWa | λ | Shapley 최소 | Shapley 최대 | 범위 |
|---:|---:|---:|---:|---:|
| 1.000 | 0.000000 | 0.122984 | 0.199597 | 0.076613 |
| 1.200 | -0.369915 | 0.121651 | 0.200756 | 0.079105 |
| 1.500 | -0.666531 | 0.120111 | 0.202116 | 0.082005 |
| 2.000 | -0.876679 | 0.118371 | 0.203691 | 0.085320 |
| 2.500 | -0.954554 | 0.117310 | 0.204685 | 0.087375 |
| 3.000 | -0.984588 | 0.116690 | 0.205289 | 0.088599 |
| 4.133 | -0.999486 | 0.116194 | 0.205806 | 0.089612 |

중요한 점은 `ΣWa=2.5`도 이미 `λ=-0.954554`로 상당히 포화에 가깝다는 것이다. 따라서 전역 6지표에 Sugeno λ를 적용하려면 목표 합계를 1.2~1.5 정도로 낮춰야 온건한 중복 보정이라고 설명할 수 있다.

## 6. 문제 2: Choquet 집계 확장 진단

### 6.1 판단

사용자가 지적한 이중 보정 문제는 방법론적으로 타당하다.

현재 결과표는 Shapley φ를 최종 Fuzzy-AHP 가중치처럼 사용한다. 그런데 여기에 다시 같은 λ-퍼지측도로 Choquet 적분을 적용하면, 동일한 중복성을 가중치 단계와 집계 단계에서 두 번 반영하게 된다.

### 6.2 현재 파일 기준

현재 엑셀 결과표는 실제 설비별 PI를 Choquet으로 계산하고 있지는 않다. 다만 `08_분석안내`에 `Choquet/Shapley`가 함께 적혀 있어, 후속 알고리즘에서 두 방법을 동시에 적용할 위험이 있다.

따라서 논문 방법론에서는 다음 중 하나만 선택해야 한다.

| 방식 | 설명 | 판단 |
|---|---|---|
| A | AHP 가중치 유지, PI 집계만 Choquet | 가능하지만 설명·검증 부담 큼 |
| B | Shapley 보정가중치 사용, PI 집계는 가중합 | 가장 안전 |
| C | Shapley 보정가중치 + Choquet 집계 | 이중 보정이므로 제외 |

현재 논문에는 방식 B가 가장 적합하다.

### 6.3 이중 보정 수치 예시

평균 정규점수를 점수로 가정하면 다음과 같이 차이가 난다.

| 방식 | 값 |
|---|---:|
| AHP 가중합 | 0.740342 |
| Shapley 가중합 | 0.719494 |
| Choquet 단독 | 0.822969 |
| Shapley×score 후 Choquet | 0.168958 |

마지막 값은 해석 가능한 PI라기보다 중복 보정이 과도하게 중첩된 결과에 가깝다. 따라서 Shapley와 Choquet을 동시에 쓰면 안 된다.

## 7. 추가 핵심 문제: 현재 Fuzzy-AHP 결과가 AHP를 실제로 보정하지 않음

현재 `AHP_vs_FAHP` 시트의 `FAHP φ` 값은 `Wa`로 구성한 λ-퍼지측도의 Shapley 값이다. 즉, AHP 가중치 `Wr`과 비교는 하고 있지만, 계산상 `Wr`이 `φ` 산정에 직접 들어가지는 않는다.

따라서 “AHP로 구한 상대중요도에 Fuzzy 분석을 결합하여 보정 중요도를 계산한다”는 예시 1의 논리와는 약간 다르다.

예시 1의 논리를 따르려면 다음처럼 AHP와 Fuzzy 값을 실제로 결합해야 한다.

| 하위지표 | AHP×Wa 정규화 | AHP×Shapley 정규화 |
|---|---:|---:|
| 투자가치 | 0.240716 | 0.242601 |
| 투자효율 | 0.114725 | 0.111235 |
| SAIDI저감 | 0.227052 | 0.231290 |
| 고장예측저감 | 0.041057 | 0.038476 |
| 안전영향 | 0.293443 | 0.298920 |
| 환경영향 | 0.083008 | 0.077479 |

이 방식은 AHP 상대가중치를 중심으로 두고, Fuzzy 절대중요도는 보정계수로 작동한다. 따라서 예시 1 기반 논리에는 현재의 순수 Shapley φ보다 더 잘 맞는다.

## 8. 수정 방향

### 8.1 가장 권장하는 방향

현재 논문에는 다음 방식이 가장 안전하다.

```text
AHP Wr 산정
→ Fuzzy 절대중요도 Wa 산정
→ 보정가중치 = Wr × Wa / Σ(Wr × Wa)
→ PI는 보정가중치 기반 가중합으로 산정
```

이 방식의 장점은 다음과 같다.

1. 예시 1의 “AHP 중요도 + Fuzzy 보정” 논리와 가장 유사하다.
2. `λ≈-1` 포화 문제가 사라진다.
3. Choquet 이중 보정 문제가 발생하지 않는다.
4. 심사위원에게 설명하기 쉽다.
5. 설비별 PI 기여도를 선형적으로 분해할 수 있다.

### 8.2 λ-퍼지측도를 꼭 쓰고 싶을 경우

Sugeno λ를 유지하려면 다음 조건이 필요하다.

1. 전역 6지표에 한 번에 적용하지 말고, 대기준 내부의 하위지표 단위로 나눠 적용한다.
2. 또는 `Wa` 합계를 1.2~1.5 수준으로 압축한다.
3. Shapley 보정가중치를 사용한다면 PI 집계는 가중합으로 유지한다.
4. Choquet 집계는 부록 또는 민감도 분석으로만 제시한다.

부모 기준 내부 2지표별 λ를 계산하면 다음과 같다.

| 부모 기준 | ΣWa | λ | 첫 번째 지표 Shapley | 두 번째 지표 Shapley |
|---|---:|---:|---:|---:|
| 경제성 | 1.450000 | -0.863424 | 투자가치 0.566667 | 투자효율 0.433333 |
| 신뢰도 | 1.350000 | -0.808081 | SAIDI저감 0.650000 | 고장예측저감 0.350000 |
| 안전·환경 | 1.333333 | -0.794834 | 안전영향 0.658333 | 환경영향 0.341667 |

전역 6지표 적용보다는 낫지만, 여전히 λ가 비교적 강한 음수이다.

## 9. 결론

사용자가 제기한 두 문제는 모두 타당하다.

1. `λ≈-1` 포화는 실제 응답자료에서 발생한다.
2. Shapley 가중치와 Choquet 집계를 동시에 쓰면 이중 보정 문제가 발생한다.
3. 현재 결과표의 `FAHP φ`는 AHP를 보정한 값이라기보다 `Wa`만으로 산출한 Shapley 가중치에 가깝다.
4. 현재 설문 구조는 기존 논문 연구모형의 `Risk` 기준과 완전히 일치하지 않는다.

따라서 본 논문에는 `AHP 주 방법 + Fuzzy 절대영향도 보정 + 가중합 PI` 방식이 가장 안전하다. Choquet 적분은 본 방법론의 주 방법으로 쓰기보다, 향후 확장 또는 부록 민감도 분석으로 두는 것이 바람직하다.

## 10. 최종 채택 방향

사용자 검토를 거쳐 최종 방법론 방향은 다음과 같이 확정한다.

| 구분 | 최종 방향 |
|---|---|
| 주 가중치 산정방법 | AHP |
| Fuzzy의 역할 | AHP 가중치의 강건성 및 판단 모호성 보조 검토 |
| Fuzzy 보정식 | `w_k^F = Wr_k × Wa_k / Σ(Wr_j × Wa_j)` |
| PI 집계 | 가중합 |
| Sugeno λ-퍼지측도 | 본문 주 방법에서 제외 |
| Choquet 적분 | 본문 주 방법에서 제외, 필요 시 부록 또는 향후 연구 |
| MCDM 기준 구조 | 경제성, 신뢰도, 안전·환경 |
| 하위지표 | 투자가치, 투자효율, SAIDI 저감, 고장예측 저감, 안전 영향 저감, 환경 영향 저감 |
| 재무 Risk | 투자가치와 투자효율에 이미 반영되므로 별도 MCDM 기준에서 제외 |

이 방향은 가중치 방법론에서 인위적으로 큰 차이를 만드는 대신, AHP와 Fuzzy 보정 결과의 유사성을 “전문가 판단의 강건성”으로 해석한다. 논문의 차별성은 가중치 산정기법 자체보다 비용 규모 보정 Scale Factor, 이종설비 통합 PI, 5개년 포트폴리오 최적화, 목표 SAIDI/Risk 민감도 분석에 둔다.
