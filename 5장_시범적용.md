# Ⅴ. 시범적용 및 검증

## 5.1 시뮬레이션 설계 및 합성 데이터 가정

### 5.1.1 합성 데이터 활용 배경

본 연구는 국내 배전사업자의 실계통 자산 데이터를 직접 확보하지 않고 합성 데이터(synthetic data)를 생성하여 시범 적용 실험을 수행하였다. 이는 다음 두 가지 이유에 근거한다.

첫째, 본 연구의 핵심 기여는 CNAIM 위험도의 예측 정확도를 높이는 것이 아니라, **다기준 평가(AHP)와 이종 자산군 통합 최적화(ILP/GA)를 결합한 의사결정 체계(framework)**를 제안하는 것이다. 따라서 입력 데이터의 현실성보다 의사결정 체계 간 상대적 성능 비교의 내적 타당성이 더 중요하다.

둘째, 합성 데이터는 **통제 실험(controlled experiment)**을 가능하게 한다. 실계통 데이터는 자산군 구성비·노후화 분포·CoF 구조 등이 복합적으로 얽혀 방법론의 순수 효과를 분리하기 어렵다. 반면 합성 데이터는 파라미터 분포를 명시적으로 설정하여 재현 가능하며(reproducible), 실험 조건을 완전히 통제할 수 있다.

본 연구의 외적 타당성 한계는 명확히 인식하며, 실계통 데이터 기반 검증은 향후 연구 과제로 남긴다.

### 5.1.4 의무교체 기준 가정

HI ≥ 7.0인 자산은 의무교체 대상으로 분류하여 예산과 무관하게 전량 교체를 강제하였다. 이는 CNAIM v2.1에서 HI 7.0 이상을 '즉각 조치 권고(Urgent)' 등급으로 규정한 것과 일치한다. 의무교체 대상은 전체 26,758기 중 5,654기(21.1%)이며, 의무교체 비용은 66,179,300천원으로 별도 예산으로 처리한다.

## 5.2 AHP 가중치 산출 결과

### 5.2.1 전문가 설문 구성

AHP 가중치 산출을 위해 전력계통 및 자산관리 분야 전문가 20명을 대상으로 설문을 실시하였다. 전문가 그룹은 표 5.6과 같이 5개 그룹으로 구성하였다.

**표 5.6  전문가 패널 구성**

| 그룹 | 직능 | 인원 | 가중치 성향 |
|-----|-----|-----:|------------|
| A (경영·재무) | CFO, 경영기획 | 1 | 경제효율성(C1) 중시 |
| B (현장 운영) | 배전 현장, 기술직 | 5 | 고객서비스(C2) 중시 |
| C (계통 계획) | 계통운영, 신뢰도 | 4 | Risk저감(C3) 중시 |
| D (자산관리) | 설비관리팀 | 4 | 균형 |
| E (규제·정책) | 규제기관, 연구원 | 6 | Risk저감(C3) 중시 |

설문지는 Classical AHP(9점 척도), Fuzzy AHP(TFN 언어척도), BWM(Best-Worst Method) 세 가지 형식으로 구성하였으며, AHP 계층 구조는 3대기준(C1: 경제효율성, C2: 고객서비스, C3: Risk저감)과 7개 하위기준으로 설계하였다.

### 5.2.2 Classical AHP 결과

전체 20명의 쌍대비교 행렬을 기하평균으로 집계한 결과, 3대기준의 일관성 비율(CR)은 0.0019로 허용 기준(CR ≤ 0.1)을 충족하였다.

**표 5.7  Classical AHP 가중치 산출 결과**

| 대기준 | 가중치 | 하위기준 | 지역 가중치 | 전역 가중치 |
|-------|------:|---------|----------:|----------:|
| C1 경제효율성 | 0.2996 | C1-1 NPV | 0.6317 | **0.1893** |
| | | C1-2 BCR | 0.3683 | **0.1103** |
| C2 고객서비스 | 0.2179 | C2-1 SAIDI저감 | 0.7889 | **0.1719** |
| | | C2-2 ENF저감 | 0.2111 | **0.0460** |
| C3 Risk저감 | 0.4825 | C3-1 재무Risk | 0.4443 | **0.2144** |
| | | C3-2 안전Risk | 0.4095 | **0.1976** |
| | | C3-3 환경Risk | 0.1463 | **0.0706** |

전문가 집단은 Risk저감(C3, 0.4825)을 가장 중요하게 평가하였으며, 경제효율성(C1, 0.2996)과 고객서비스(C2, 0.2179)가 뒤를 이었다. 이는 배전설비 투자 의사결정에서 리스크 관리가 최우선 고려 요소임을 반영한다.

### 5.2.3 Fuzzy AHP 결과

언어형 불확실성을 TFN(삼각 퍼지 수)으로 변환한 후 Chang(1996)의 확장분석법을 적용하였다. 결과는 Classical AHP와 유사한 경향을 보였으나, C2-1 SAIDI저감의 퍼지 가중치(0.2137)가 Classical AHP(0.1719) 대비 소폭 높게 산출되었다. 이는 현장 전문가 그룹이 정전 시간 저감에 높은 중요도를 부여하는 언어적 표현이 TFN 변환 과정에서 보다 민감하게 반영된 결과로 해석된다.

### 5.2.4 BWM 결과

Rezaei(2015)의 Best-Worst Method를 적용한 결과, 최적 일관성 지표(ξ*)는 대기준 수준에서 0.4696을 기록하였다. ξ* = 0이 완전 일관성을 의미하므로, 이 값은 전문가 판단 간 일정 수준의 불일치가 존재함을 나타낸다. 이는 20명의 다양한 직능 그룹이 참여한 데 따른 자연스러운 결과로, 3종 방법론 중 가중치 분산이 가장 크게 나타났다.

### 5.2.5 3종 방법론 가중치 비교

### 5.3.2 혼합가치 (Mixed Value) 산출

제안 방법론의 목적함수인 혼합가치 $V_j$는 7개 하위기준별 Min-Max 정규화 점수에 AHP 전역 가중치를 적용하여 산출한다.

$$V_j = \sum_{i=1}^{7} w_i \cdot s_{ij}$$

여기서 $s_{ij}$는 자산 $j$의 하위기준 $i$ 점수(0~1 정규화), $w_i$는 Classical AHP 전역 가중치이다. 각 점수의 원시값(raw value)은 표 5.9와 같다.

**표 5.9  하위기준별 점수 원시값 정의**

| 하위기준 | 기호 | 원시값 (정규화 전) |
|---------|-----|----------------|
| C1-1 NPV | $r_{NPV,j}$ | 투자가치NPV_천원 |
| C1-2 BCR | $r_{BCR,j}$ | NPV / 교체비용 |
| C2-1 SAIDI | $r_{SAIDI,j}$ | PoF$_j$ × MTTR$_j$ × CoF_{NP,j} |
| C2-2 ENF | $r_{ENF,j}$ | PoF$_j$ × CoF_{NP,j} |
| C3-1 재무Risk | $r_{FR,j}$ | PoF$_j$ × CoF_{F,j} |
| C3-2 안전Risk | $r_{SR,j}$ | PoF$_j$ × CoF_{S,j} |
| C3-3 환경Risk | $r_{ER,j}$ | PoF$_j$ × CoF_{E,j} |

눈에 띄는 첫 번째 발견은 Greedy(B1, B2)와 ILP(B3, B4)의 결과가 각각 거의 동일하다는 점이다. 자산유형별로 예산을 분리하면 각 자산유형 내 최적화 문제의 규모가 대폭 축소되어, Greedy 방식으로도 ILP와 동등한 선택 집합을 도출하게 된다. 이는 **자산군 분리 배분 구조 하에서는 최적화 기법의 고도화가 실질적 효과를 발휘하지 못함**을 보여 주며, 관행적 접근법의 구조적 한계를 실증한다.

#### 주요 관찰 (2): P1의 일관된 우위

## 5.6 토의

### 5.6.1 제안 방법론의 기여

본 실험 결과는 제안 방법론(다기준 평가 + 이종 자산군 통합 ILP)이 기존 관행(자산유형별 분리 단일기준 선택) 대비 실질적인 성과 개선을 달성함을 보여 준다.

- **포트폴리오 최적화 효과**: 전체 예산을 통합 최적화함으로써 자산유형 간 예산 재배분이 가능해지고, 단위 혼합가치가 높은 자산에 자원이 집중된다.
- **다기준 통합의 가치**: 단일 기준(NPV 또는 Risk)으로는 포착되지 않는 안전·환경·신뢰도 차원의 가치가 혼합가치 지표를 통해 의사결정에 반영된다.
- **방법론 강건성**: 예산 수준이 3%~7%로 변화하는 상황에서도 P1의 우위가 일관되게 유지된다.

### 5.6.2 한계 및 향후 연구

본 연구의 한계는 다음과 같다.

첫째, **합성 데이터 의존성**이다. 실계통 데이터에서는 자산유형별 CoF 구조, 실제 고장 이력, 현장 제약 조건이 본 연구의 가정과 다를 수 있다. 실계통 검증이 후속 연구로 필요하다.

둘째, **전문가 패널의 대표성**이다. 본 연구에서는 20명의 가상 전문가 설문 데이터를 사용하였다. 실제 전문가 5~9명을 대상으로 한 설문 수행 및 검증이 필요하다.

셋째, **ILP 규모 제약**이다. 26,758기 규모에서 60초 이내 ILP 최적 해 도달이 불가능하여 Greedy fallback이 적용되었다. 상용 ILP 솔버(Gurobi, CPLEX) 또는 분기한정법 고도화가 필요하다.

넷째, **단기 관점**이다. 본 연구는 1년 단위 투자계획을 대상으로 하였다. 다기간(multi-period) 최적화 모델로의 확장이 향후 연구 방향이다.

## 5.7 소결

본 장에서는 26,758기 규모의 합성 배전설비 데이터에 제안 방법론을 시범 적용하고, 기존 관행 4종(B1~B4)과 비교 분석하였다. 핵심 결과를 요약하면 다음과 같다.

1. 자산유형별 분리 배분(B1~B4)에서는 최적화 기법의 고도화(ILP 도입)가 Greedy 대비 실질적 성과 차이를 만들지 못한다.
2. 제안 방법론(P1: 통합AHP-ILP)은 관행 최선(B4) 대비 총NPV +8.1%, BCR +8.0%, Risk저감 +4.7%, 혼합가치합 +10.3%의 일관된 개선을 달성한다.
3. 이 성과 개선은 예산 수준(3%, 5%, 7%)에 무관하게 강건하게 유지되어 제안 방법론의 적용 범위가 넓음을 확인하였다.
4. GA(P2)는 ILP 대비 성과가 소폭 낮지만 대규모 실계통 적용 시 실용적 대안이 될 수 있다.
