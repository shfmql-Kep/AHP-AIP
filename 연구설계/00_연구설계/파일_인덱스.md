# 파일 인덱스

이 문서는 `연구설계` 폴더 안에서 논문 검토와 작성에 우선적으로 확인해야 할 파일을 정리한 것이다.

## 1. 연구설계와 작성 기준

| 파일 | 역할 |
|---|---|
| `00_연구설계/연구설계_최종.md` | 연구 범위, 핵심 기여, 시뮬레이션 구조, 해석 원칙 |
| `00_연구설계/AGENTS.md` | 작업 언어 및 기본 응답 규칙 |
| `00_연구설계/CLAUDE.md` | 기존 작업 규칙 참고 |

## 2. 논문 원고

| 폴더 | 역할 |
|---|---|
| `01_논문_장별` | 최신 장별 논문 원고 |

우선 검토 순서는 다음과 같다.

```text
00_목차_및_작성정보.md
02_제1장_서론.md
03_제2장_이론적_배경.md
05_제4장_제안_방법론.md
06_제5장_시범적용_및_결과_분석.md
07_제6장_결론_및_향후_연구.md
08_참고문헌.md
```

## 3. 입력 데이터

| 파일 | 역할 |
|---|---|
| `02_입력데이터/input_assets.xlsx` | 설비별 입력 자산 데이터 |
| `02_입력데이터/cnaim_params.xlsx` | PoF 및 HI 변환 관련 파라미터 |
| `02_입력데이터/pof_5yr_output.xlsx` | 2026~2030년 PoF, Risk, 투자가치, SAIDI 등 최적화 입력 |

## 4. 설문조사 자료

| 파일 | 역할 |
|---|---|
| `03_설문조사/통합설비_FuzzyAHP_설문지_v2_예시1구조.xlsx` | 설문지 양식 |
| `03_설문조사/통합설비_FuzzyAHP_설문지_v2_응답완료_20명.xlsx` | 전문가 20명 응답지 |
| `03_설문조사/설문결과_퍼지측도_진단보고서.md` | Fuzzy 측도 및 설문 구조 진단 |
