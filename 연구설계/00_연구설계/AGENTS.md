# 글로벌 규칙

## 언어 규칙

- 모든 응답은 한국어로 작성한다.
- 코드 주석도 한국어로 작성한다.
- 오류 메시지 설명, 분석 내용 등 모든 텍스트는 한국어로 한다.
- 단, 코드 자체(변수명, 함수명 등)는 영어를 유지한다.

## 박사논문 작성 규칙

- 논문 본문은 박사논문 예시의 문체를 기준으로 작성한다.
- 사례 고찰, 이론적 배경, 선행연구 분석에서는 “○○ 자료에 따르면”, “○○에 소개된 자료에서는”처럼 보고서 요약문 또는 발표자료 해설문처럼 보이는 표현을 사용하지 않는다.
- 참고자료의 내용을 활용할 때는 자료명을 문장 앞에 세워 설명하지 말고, 해당 사례·기관·방법론의 내용으로 자연스럽게 서술한 뒤 문장 끝에 참고문헌 번호를 표시한다.
- 비공개 보고서, 발표자료, 기사형 자료를 활용하더라도 논문 본문에서는 박사논문 사례 고찰 문체로 재구성한다.
- 사용자가 “그대로 가져오라”고 지시한 경우에도 원문 문장 복제보다 논문 문체에 맞는 재서술을 우선한다. 단, 핵심 내용·수치·절차·용어는 누락하지 않는다.
- 한글 텍스트가 깨지거나 추출 품질이 낮을 경우 자의로 해석하지 않고, 인코딩·추출 도구·원본 렌더링 확인 등 복구 방법을 먼저 찾는다.
