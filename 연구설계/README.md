# 연구설계 폴더 사용안내

이 폴더는 논문 검토와 본문 작성을 위해 필요한 핵심 자료만 모은 작업용 패키지이다.  
앞으로 논문 구조, 방법론, 시뮬레이션 결과, 설문 기반 PI 산정 내용을 검토할 때에는 우선 이 폴더를 기준으로 확인한다.

## 1. 폴더 구성

| 폴더 | 내용 | 사용 목적 |
|---|---|---|
| `00_연구설계` | 최종 연구설계, 작업 규칙 | 논문 방향과 용어 기준 확인 |
| `01_논문_장별` | 장별 논문 원고 | 실제 본문 검토 및 수정 |
| `02_입력데이터` | 입력 자산 데이터, CNAIM 파라미터, 5개년 PoF 출력 | 시뮬레이션 입력 검토 |
| `03_설문조사` | 설문지, 전문가 20명 응답지, 진단보고서 | AHP/Fuzzy 보정 근거 확인 |
| `04_알고리즘/python` | 데이터 생성, PI 보조 처리, 후처리 스크립트 | 입력데이터 및 결과 후처리 |
| `04_알고리즘/matlab` | PI 산정, 최적화, 민감도 분석 MATLAB 모델 | 핵심 시뮬레이션 재현 |
| `05_PI_산출결과` | 설비 단위 PI, 통합 PI, BWM/MCDM 비교 결과 | PI 산정 결과 검토 |

| `07_그림자료` | 생성 그림 및 주요 원자료 그림 | 본문 그림 삽입 |

| `09_삭제후보` | 삭제 후보 목록 | 원본 작업공간 정리 전 검토 |

## 2. 현재 연구설계 기준

본 연구의 핵심 기여는 다음 네 가지로 정리한다.

1. Risk 산정 결과를 활용한 투자가치 산정
2. AHP와 Fuzzy 보정을 결합한 다기준 PI 산정
3. 설비 단위 PI와 통합 PI를 활용한 5개년 포트폴리오 최적화
4. 예산, 물량, SAIDI, Risk 총량, 운영목표 가중치 변화에 대한 민감도 및 강건성 분석

## 3. 시뮬레이션 해석 원칙

- 메인 비교는 Greedy, ILP, GA 결과를 중심으로 해석한다.
- NSGA-II는 대표 결과가 아니라 확장 다목적 탐색 결과로만 사용한다.
- 투자가치 최적화에서는 투자비용 최소화를 별도 목적함수로 병렬 적용하지 않는다.
- 대표 통합 방식은 `2단계 통합 PI 최적화 방식`이다.
- `사전 통합 PI 방식`은 비교 시나리오로만 사용한다.
- BWM은 독립 설문 기반 결과가 아니라 AHP 응답 기반 재구성 비교군으로 설명한다.

## 4. 주의사항

- 이 폴더는 원본 자료를 복사하여 구성한 작업용 폴더이다.
- 원본 삭제는 아직 수행하지 않았다.
- 삭제 후보는 `09_삭제후보/삭제후보_검토목록.md`를 확인한 후 별도로 확정한다.
- 설문 응답지와 비공개 성격의 분석자료가 포함되어 있으므로 외부 공유 전 반드시 확인한다.
