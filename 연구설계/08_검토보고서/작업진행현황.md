# 작업진행현황

작성일: 2026-06-20

## 1. 현재 상태

설문지 관련 작업은 사용자 요청에 따라 전면 리셋한 뒤, `Survey` 폴더의 새 설문지와 20명 응답 결과를 재검토했다.

재검토 결과, Sugeno λ-퍼지측도 기반 Shapley 가중치는 `λ≈-1` 포화와 Choquet 이중 보정 위험이 있어 본문 주 방법에서 제외하기로 했다. 최종 방향은 `AHP 주 방법 + Fuzzy 절대영향도 보정 강건성 검토 + 가중합 PI`이다.

단, CNAIM 기반 PoF 데이터와 입력 데이터셋은 유지한다. 과거 단일지표 최적화 결과는 참고 이력으로만 보관하고, 논문 본문 확정값은 최신 Matlab 재실행 결과를 기준으로 반영한다.

## 4-1-1. PoF 출력 파일 보강

로컬 PI 6개 하위지표 산출을 위해 `data/pof_5yr_output.xlsx`의 `pof_5yr` 시트에 안전·환경 CoF 관련 컬럼을 추가했다.

| 추가 컬럼 | 의미 | 활용 |
|---|---|---|
| `cof_safety_kkrw` | 자산별 안전 CoF | 안전 영향 지표 |
| `cof_environment_base_kkrw` | 입력 데이터의 기본 환경 CoF | 검증·감사용 |
| `cof_environment_adjusted_kkrw` | 수계 여부와 규모 보정을 반영한 환경 CoF | 환경 영향 지표 |

재생성 결과 `pof_5yr` 시트는 21,893행, 57열이며, 추가 CoF 컬럼의 결측값은 없다.

## 4-1-2. 로컬 PI 산출 공식 개선

통합설비 PI는 아직 수정하지 않고, 개별 설비 로컬 PI만 6개 하위지표 기반으로 재산정했다. 산출 코드는 `code/local_pi_matlab.m`이며, 결과 파일은 `outputs/local_pi_matlab.xlsx`이다.

| 구분 | 적용 내용 |
|---|---|
| 주 PI | AHP 기반 `local_pi_ahp` |
| 보조 PI | Fuzzy 절대영향도 보정 기반 `local_pi_fuzzy_adjusted` |
| Fuzzy 보정식 | `w_k^F = Wr_k × Wa_k / Σ(Wr_j × Wa_j)` |
| 정규화 | 전체 자산·전체 연도 기준 P95 robust scaling, 0~1 상한 |
| 안전 지표 | `pof_YYYY × cof_safety_kkrw` |
| 환경 지표 | `pof_YYYY × cof_environment_adjusted_kkrw` |
| 고장예측 지표 | `pof_YYYY` |

가중치 산정 결과는 다음과 같다.

| 하위지표 | AHP 가중치 | Fuzzy 보정 가중치 |
|---|---:|---:|
| 투자가치 | 0.225110 | 0.240716 |
| 투자효율 | 0.129016 | 0.114725 |
| SAIDI저감 | 0.203753 | 0.227052 |
| 고장예측저감 | 0.057898 | 0.041057 |
| 안전영향 | 0.263331 | 0.293443 |
| 환경영향 | 0.120893 | 0.083008 |

검증 결과 AHP 하위지표 가중치 합계와 Fuzzy 보정가중치 합계는 모두 1.000000이며, `local_pi_asset_wide`는 21,893행, `local_pi_asset_year`는 109,465행으로 생성되었다.

## 7. 설문 재설계 원칙

새 설문 및 가중치 분석은 다음 원칙으로 정리한다.

1. 전문가에게 묻는 내용과 데이터로 계산하는 내용을 명확히 분리한다.
2. 설문 문항 수를 최소화해 응답 신뢰도를 높인다.
3. AHP를 주 가중치 산정방법으로 사용한다.
4. 설비유형 정규화는 통합설비 최적화에 꼭 필요한 경우에만 포함한다.
5. Fuzzy 절대영향도는 `Wr×Wa` 보정가중치로만 사용하고, λ-퍼지측도와 Choquet 적분은 본문 주 방법에서 제외한다.
6. AHP와 Fuzzy 보정 결과가 유사하면 이를 가중치 강건성으로 해석한다.
