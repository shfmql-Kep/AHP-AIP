### 2.2 표지 형식
- 최상단: `博士學位論文` (한자)
- 한국어 제목 (신명조 계열)
- 영문 제목 (이탤릭 허용)
- `指導敎授 姜 亨 求` (한자 표기)
- `20XX年 X月`
- `韓國交通大學校 大學院`
- `電 氣 工 學 科`
- `저자명 (한자)`

### 2.3 장 번호 체계 (예시4·5 공통)

```
Ⅰ. (또는 I.)  → 장 (Chapter)
  1.1 / 2.1    → 절
    1.1.1       → 항
```

- 장: 로마 숫자 대문자 (Ⅰ, II, III, IV, V, VI...)
- 절: 아라비아 숫자 (1.1, 1.2 ... / 2.1, 2.2 ...)
- 항: 소수점 세 자리 (1.1.1, 1.1.2 ...)
