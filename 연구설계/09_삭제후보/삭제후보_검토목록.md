# 삭제 후보 검토목록

이 목록은 실제 삭제를 수행한 결과가 아니라, 원본 작업공간 정리를 위해 검토할 후보를 정리한 것이다.  
삭제는 사용자가 최종 확인한 뒤에만 수행한다.

## 1. 루트 폴더의 중간 산출물 후보

다음 파일들은 최신 장별 원고 또는 `연구설계` 폴더에 필요한 내용이 반영되어 있다면 삭제 후보로 볼 수 있다.

| 파일 | 판단 |
|---|---|
| `논문_통합본.md` | 중간 통합본. 최신 작업은 `논문_장별` 기준이므로 삭제 후보 |
| `논문_통합본_수정본.md` | 중간 통합본. 사용자가 장별 파일만 수정하기로 했으므로 삭제 후보 |
| `tmp_simulation_summary_extract.md` | 임시 추출자료. 최신 결과 파일이 있으면 삭제 후보 |
| `MCDM_PI_최적화_결과요약.md` | 최신 제5장 또는 결과 엑셀에 반영되었으면 삭제 후보 |
| `pptx_slides.txt` | 발표자료 텍스트 추출본으로 보이며, 논문 본문 작성에는 낮은 우선순위 |
| `임시자료_목차_문헌조사_용어정리.md` | 핵심 내용 반영 여부 확인 후 삭제 후보 |
| `임시자료_에셋상태데이터수정.md` | 핵심 내용은 연구설계 한계와 입력데이터 설명에 반영됨. 삭제 후보 |
| `임시자료_자산가액기반_예산제약.md` | 예산제약 근거 반영 후 삭제 후보 |

## 2. 오래된 시뮬레이션 결과 폴더 후보

최신 본문 시뮬레이션 결과는 다음 폴더를 기준으로 한다.

```text
outputs/simulation_chapter_v2_matlab_20260621_183538
```

따라서 아래와 같은 이전 실행 폴더는 삭제 후보이다.

```text
outputs/simulation_chapter_v2_matlab_20260620_171833
outputs/simulation_chapter_v2_matlab_20260620_183432
outputs/simulation_chapter_v2_matlab_20260620_185148
outputs/simulation_chapter_v2_matlab_20260620_191232
outputs/simulation_chapter_v2_matlab_20260620_191348
outputs/simulation_chapter_v2_matlab_20260620_191454
outputs/simulation_chapter_v2_matlab_20260620_192011
outputs/simulation_chapter_v2_matlab_20260620_192335
outputs/simulation_chapter_v2_matlab_20260620_202659
```

단, 과거 결과를 논문 수정 이력 또는 비교 근거로 보관하려면 삭제하지 않는다.

## 3. 오래된 민감도 분석 결과 폴더 후보

최신 민감도 분석 결과는 다음 폴더를 기준으로 한다.

```text
outputs/sensitivity_analysis_fast_20260621_113239
```

따라서 아래 폴더는 삭제 후보이다.

```text
outputs/sensitivity_analysis_fast_20260621_100602
outputs/sensitivity_analysis_fast_20260621_100613
```

## 4. 개발·캐시 성격 폴더 후보

다음 폴더는 논문 작성 패키지에는 포함하지 않았으나, 원본 정리 시 삭제 여부를 별도로 판단한다.

| 폴더 | 판단 |
|---|---|
| `tmp` | 임시 추출물과 검토본. 필요한 DOCX 추출본이 남아 있으면 보관, 아니면 삭제 후보 |
| `.claude` | 에이전트 작업 흔적. 필요 없으면 삭제 후보 |
| `node_modules` | 의존성 폴더. 재설치 가능하면 삭제 후보 |
| `oldver` | 과거 MATLAB 알고리즘 학습용. 논문 재현에 필요 없으면 별도 보관 또는 삭제 후보 |
| `research_cache` | 공개 참고자료 캐시. 필요 시 보관 |
| `private_research_cache` | 비공개 참고자료 캐시. 외부 공유 금지, 삭제 여부 신중히 판단 |

## 5. 삭제 보류 권장

다음은 현재 삭제하지 않는 것이 좋다.

| 항목 | 이유 |
|---|---|
| `data` | 입력 데이터 원본 |
| `Survey` | 설문 원본과 응답지 |
| `code` | 원본 알고리즘 |
| `outputs/local_pi_matlab.xlsx` | PI 산정 결과 원본 |
| `outputs/integrated_pi_matlab.xlsx` | 통합 PI 산정 결과 원본 |
| `outputs/bwm_pi_matlab.xlsx` | BWM 비교 결과 원본 |
| `outputs/simulation_chapter_v2_matlab_20260621_183538` | 최신 본문 시뮬레이션 결과 |
| `outputs/sensitivity_analysis_fast_20260621_113239` | 최신 민감도 분석 결과 |
| `논문_장별` | 최신 논문 본문 원고 |
| `참고자료`, `박사논문 예시` | 본문 작성 시 계속 참조 가능성이 높음 |

## 6. 권장 정리 순서

1. `연구설계` 폴더 안의 파일이 충분한지 확인한다.
2. 최신 제5장 결과표와 민감도 결과가 정상적으로 열리는지 확인한다.
3. 삭제 후보 중 루트 임시파일부터 삭제한다.
4. 오래된 출력 폴더는 압축 백업 후 삭제한다.
5. `tmp`, `.claude`, `node_modules`는 마지막에 정리한다.
