### 2.1 Classical AHP (Analytic Hierarchy Process)

#### 2.1.1 핵심 원리 및 절차 (NotebookLM 분석 기반)

Thomas Saaty(1970년대)가 수학과 심리학을 결합하여 개발한 의사결정 방법론.

**4단계 절차:**
1. **계층 구조화**: 목표(Goal) → 평가기준(Criteria) → 대안(Alternatives) 3단계
2. **쌍대비교**: 1-9 척도(1=동등, 3=보통, 5=강함, 7=매우강함, 9=절대)로 기준 간 상대적 중요도 비교
3. **가중치 도출**: 비교행렬 열 합으로 정규화 → 행별 평균 = 최종 가중치
4. **일관성 검증**: CI = (λ_max - n)/(n-1), CR = CI/RI → **CR ≤ 0.1** 충족 필수

#### 2.1.2 본 연구 적용 방식 — SAW+AHP 하이브리드
- **기준 점수(score)**: CNAIM Monetised Risk, SAIDI 기여도 등 객관 데이터에서 산출
- **기준 가중치(weight)**: 전문가 쌍대비교 설문으로 결정
- 포지셔닝: "Simple Additive Weighting(SAW) + AHP 가중치" 방식으로 학술 명시 필요
- 기준값 정규화 필수: Min-Max 또는 Z-score (단위 이질성: £ vs 시간 vs 비율)

#### 2.1.3 장단점

| 장점 | 한계 |
|------|------|
| 일관성 검증 내장 (TOPSIS·ELECTRE에는 없음) | 기준 독립성 가정 → 상호의존 반영 불가 |
| 유·무형 기준 동시 처리 가능 | 1-9 척도의 주관성·모호성 |
| 계층적 분해로 복잡 문제 구조화 | 대규모 기준 수에서 쌍대비교 증가 (n(n-1)/2) |
| 투명성·추적가능성 → 감사·설명책임 | 인간 판단 불확실성 완전 미반영 |

### 2.2 Fuzzy AHP (FAHP)

#### 2.2.1 개요 및 동기
Classical AHP의 핵심 한계: **고정 수치(crisp number)로 인간 판단의 모호성 미반영**. 전문가가 "7"이 아닌 "6~8 사이 어딘가"라고 느낄 때, 이를 수용할 수단 없음.

#### 2.2.2 삼각 퍼지수(Triangular Fuzzy Number, TFN) 방식

퍼지수 표현: **(l, m, u)** — 최솟값, 최빈값(가장 가능성 높은 값), 최댓값

- 예) Classical AHP에서 "7" → FAHP에서 **(6, 7, 8)**
- 멤버십 함수: l ≤ x ≤ m에서 선형 증가, m ≤ x ≤ u에서 선형 감소

#### 2.2.3 주요 가중치 산출 방법 2종

**① Chang's Extent Analysis (1996)**
- 각 기준의 퍼지 종합 범위(fuzzy synthetic extent) 계산
- 퍼지 수 간 크기 비교(degree of possibility)로 가중치 결정
- 빠르고 간단하나, **일부 기준 가중치가 0이 되는 문제** 지적 (van Laarhoven & Pedrycz 비판)

**② 기하평균 기반 방법 (Geometric Mean)**
- 각 행의 퍼지 기하평균 계산
- 역퍼지화(defuzzification)로 BNP(Best Non-fuzzy Performance) 값 도출
- **더 안정적이고 권장되는 방법** (문헌 다수 지지)

#### 2.2.4 5단계 절차
1. 기준 계층 구조 설정
2. TFN 쌍대비교 행렬 구성 (전문가 설문)
3. 퍼지 기하평균 또는 Extent Analysis로 퍼지 가중치 계산
4. 역퍼지화 → 확정 가중치 도출
5. 가중치 정규화

#### 2.2.5 전력설비 적용 사례
- **Guo et al. (2025, PLoS One)**: 변압기 유지보수에서 FAHP 적용, OPR(0.2064), ESR(0.0836), DDR(0.0783), OMCR(0.6316) 가중치 도출. 주관적 가중치(80%) + 객관적(CRITIC+Entropy, 20%) 혼합. **40~60% 유지비 절감** 효과
- **Varshney et al. (2024, Scientific Reports)**: 전력계통 자동발전제어(AGC)에 FAHP 적용, 주파수편차·면적제어오류·계통연계오류 가중 목적함수 산출. W₁=0.7791, W₂=0.1706, W₃=0.0503
- **Peng et al. (2021, Frontiers)**: 토지수용 갈등 리스크 평가에 FAHP 적용, 합법성·타당성·통제가능성 계층 분석

#### 2.2.6 장단점

| 장점 | 한계 |
|------|------|
| 불확실성·모호성 수용 → 전문가 수용성 향상 | 연산 복잡도 증가 |
| 언어적 표현 ("약간 중요", "매우 중요") 수용 | Extent Analysis 시 일부 가중치 0 문제 |
| 이해관계자 설득력 향상 | 역퍼지화 방법 선택에 따라 결과 상이 |
| CR 검증 유지 가능 | Classical AHP보다 설문 설계 복잡 |

### 2.3 Delphi-AHP 통합 방법론

#### 2.3.1 Delphi 방법론 개요
**1969년 RAND Corporation**이 기술 예측을 위해 개발. 전문가 집단의 익명 반복 설문으로 합의를 도출하는 방법.

**핵심 특징 (NotebookLM 분석):**
- **익명성**: 카리스마·권위자 편향 제거
- **반복성**: 라운드마다 타 전문가 의견·근거 피드백 → 자기 수정 기회
- **합의 지향**: 최종적으로 전문가 집단의 수렴된 견해 도출
- **적용 조건**: 점진적·진화적 변화가 예상되는 영역 적합 (급격한 불연속 변화에는 부적합)

#### 2.3.2 Delphi-AHP 4단계 통합 절차

**Phase 1 (Delphi Round 1)**: 개방형 설문 → 평가기준 후보 도출
**Phase 2 (Delphi Round 2~3)**: 합의도 기반 기준 정제

- **CVR(Content Validity Ratio) 기준**: CVR = (n_e - N/2) / (N/2)
  - 18명 전문가 패널 기준: CVR ≥ 0.444 합격
  - 합의 임계값: 전문가 ≥75~80% "매우 중요" 평가, IQR ≤ 1

**Phase 3 (AHP)**: 확정 기준으로 쌍대비교 → 가중치 산출
**Phase 4 (검증)**: 일관성 확인, 민감도 분석

#### 2.3.3 적용 사례

**Maletič et al. (2025, J. Health Org. Mgmt.)**: 헬스케어 자산관리시스템 장벽 식별
- 전문가 30명(26명 완료), 3 Delphi 라운드 + AHP 단계
- 23개 장벽 → 5개 차원(전략·인적자원·맥락·구조·절차)으로 분류
- 1위 장벽: 자격 인력 부족(9.8%), 업무 과부하(8.5%), 리더십 결핍(7.6%)
- **전력유틸리티 시사점**: 헬스케어와 구조적 유사성 — 인력 제약·리더십·자원 부족이 공통 장벽

**Pirdashti & Omidi (Semantic Scholar)**: AHP-Delphi MCDM 모델 환경 의사결정 적용
**Delphi+AHP+PROMETHEE**: 재생에너지 정책에서 기술·경제·사회·환경 기준 통합 평가 (2020)
**통합 Delphi-AHP (2024, MDPI Systems)**: Industry 4.0 유지보수 역량 우선순위 — 15명 전문가, 3 Delphi 라운드, 29개 지표 최종 채택

#### 2.3.4 장단점

| 장점 | 한계 |
|------|------|
| 전문가 편향·집단사고 방지 | 시간 소요 많음 (라운드 반복) |
| 기준 선정 과정 자체의 학술 기여 | 전문가 패널 구성 어려움 |
| CVR로 기준 타당성 검증 | 전문가 중도 이탈 위험 |
| AHP와 시너지: 기준 도출(Delphi) + 가중화(AHP) | 진행자 역량에 결과 의존 |

### 2.4 ANP (Analytic Network Process)

#### 2.4.1 개요
AHP의 계층 독립성 가정을 완화한 일반화 모델. Saaty가 AHP 이후 발전.

**핵심 차이**: AHP는 기준 간 독립성 가정 → ANP는 **기준 간 상호의존성·피드백 반영**

#### 2.4.2 슈퍼매트릭스 구조

```
           C1    C2    C3
    C1  [W11  W12  W13]
W = C2  [W21  W22  W23]  ← 슈퍼매트릭스
    C3  [W31  W32  W33]
```

- Wij: i가 j에 미치는 영향 가중치 (AHP 비교로 산출)
- **열 합산 수렴 조건**: 열 합 ≤ 1 (가중 슈퍼매트릭스)
- **극한 슈퍼매트릭스**: W^∞로 수렴 → 최종 ANP 가중치

#### 2.4.3 AHP vs ANP 사용 조건

| 조건 | AHP | ANP |
|------|-----|-----|
| 기준 간 독립 | ✅ 적합 | 과잉 설계 |
| 기준 간 상호영향 | ⚠️ 부정확 | ✅ 적합 |
| 대안→기준 피드백 | ❌ 불가 | ✅ 가능 |
| 계산 복잡도 | 낮음 | 높음 |
| 전문가 설문 부담 | 중간 | 높음 (더 많은 비교) |
| 소프트웨어 필요 | Excel 가능 | SuperDecisions 권장 |

#### 2.4.4 전력/인프라 적용 사례
- **AHP/ANP 태양열 발전소 투자 선택 (Elsevier Energy, 2014)**: 기술·경제·환경·사회 기준의 상호영향 반영
- **복잡한 그리드 투자 (Frontiers Energy Research, 2022)**: 인덱스 시스템 변화 시 슈퍼매트릭스 재계산 필요 → ANP 채택
- **ANP 적용 리뷰 (J. Multi-Criteria Decision Analysis, 2024)**: 경제·금융·경영 분야 확산, 원자력 부지 선정에도 활용

#### 2.4.5 본 연구 ANP 적용 고려사항
배전설비 투자 기준 간 의존성 예시:
- CNAIM Risk ↔ 공급신뢰도(SAIDI): Risk 높으면 SAIDI 기여 가능성도 높음
- 사회적 영향 ↔ 운영제약: 민감 지역은 정전 가능 시간도 제한적

**→ ANP가 AHP보다 이론적으로 정확하나, 추가 설문 부담과 복잡도 증가를 Trade-off로 고려 필요**

- **시사점**: 설문 부담 감소, 대규모 자산군 평가에 적합

- **한계**: 2영역 계통만 적용, 다영역 확장 미검증

#### 2.5.4 AI-AHP의 본 연구 구현 방향 (사례조사 결과)

**권장 방향 (합성 데이터 활용 가능):**

### 2.6 AHP + 최적화 결합 패턴

#### 2.6.1 핵심 패러다임

**2단 구조 (본 연구 채택):**
```
Stage 1: AHP → 기준 가중치 w_i + 자산 종합점수 S_j
Stage 2: 최적화 → 예산·자원 제약 하 투자 포트폴리오 X* 도출
```

#### 2.6.2 최적화 모델 비교

**① 0-1 Knapsack**

```
max  Σ S_j · x_j
s.t. Σ C_j · x_j ≤ Budget
     x_j ∈ {0,1}
```

- 가장 단순·직관적
- 단일 예산 제약에 적합
- **배전망 보전 직접 선례**: "A knapsack problem approach for optimal allocation of maintenance resources on electric power distribution networks" (ResearchGate)

**② 0-1 Integer Linear Programming (ILP)**

```
max  Σ S_j · x_j
s.t. Σ C_j · x_j ≤ Budget           (예산)
     Σ T_j · x_j ≤ T_max             (정전시간/인력)
     x_k = 1  ∀k ∈ Mandatory         (규제 의무)
     x_j ∈ {0,1}
```

- 다중 제약 처리 가능 → **배전설비 현실 반영에 가장 적합**
- Lee & Kim (2000): AHP + ILP 정보시스템 프로젝트 선택 방법론 확립

**③ Goal Programming (GP)**

```
min  Σ w_k(d⁺_k + d⁻_k)
s.t. Σ S_j · x_j + d⁻₁ - d⁺₁ = Target₁  (점수 목표)
     Σ C_j · x_j + d⁻₂ - d⁺₂ = Budget₂   (비용 목표)
     x_j ∈ {0,1}
```

- 다목적 절충에 유리
- 비용 vs 신뢰도 vs 안전을 동시에 고려할 때
- **AHP + GP 자산배분 (SciRP)**: AHP로 기준 가중 → GP로 최적 포트폴리오

#### 2.6.3 선행 연구 사례

| 연구 | 방법 | 도메인 | 결과 |
|------|------|--------|------|
| PMI (Vargas 2010) | AHP + 0-1 ILP | 프로젝트 포트폴리오 | AHP 가중치 → ILP 목적함수 통합 표준화 |
| Bahloul & Abid (SSRN) | AHP + GP | 국제 포트폴리오 | 투자장벽 7종 AHP 가중 → GP 최적화 |
| Vaezi et al. (PLoS One, 2019) | Knapsack + 불확실성 | 주식 포트폴리오 | 구간값 Knapsack, 정수 주식 수량 결정 |

### 3.2 핵심 알고리즘

**PoF (Probability of Failure):**
```
PoF = K · e^(C·H) ≈ K · [1 + (C·H) + (C·H)²/2! + (C·H)³/3!]
```
- K: 관측 고장통계 기반 보정계수
- C: 곡선 형태 결정 파라미터
- H: Health Score (나이, 운전조건, 환경계수, 진단데이터 기반)

**CoF (Consequence of Failure):**
```
CoF(£) = Network cost + Safety cost + Environmental cost + Financial cost
```

**Monetised Risk:**
```
Risk = PoF × CoF  [£/year]
```

**수명 예시** (변압기):
- 25년 → PoF = 0.01%/년 (양호 조건)
- 55년 → PoF = 0.03%/년 (지수적 증가)

### 3.5 CNAIM의 한계 — 본 연구의 출발점

1. **단일 화폐 차원**: 다중 기준(신뢰도·사회영향·운영제약) 미반영
2. **정성 변수 배제**: 현장 운영자 경험, 민원, 지역 특성 미포함
3. **동일 Risk 설비 간 우선순위 모호**: Monetised Risk가 같을 때 구분 불가
4. **역사 데이터 의존**: 새 자산군·신규 환경에서 PoF 추정 불확실
5. **실행 가능성 미반영**: 예산·정전시간·인력 등 현장 제약 미포함

→ **본 연구**: AHP(5종 방법론 비교) + 최적화로 위 5가지 한계를 보완

## 4. AIP 현황 및 현장 GAP 분석

### 4.1 AIP 현황
- **정의**: 데이터 기반 장기 자본계획, 자산 생애주기 전체 투자 대안 평가
- **핵심 시나리오**: REPLACE / REFURBISH / DO NOTHING (비용+Risk 비교)
- **국제 표준**: ISO 55000 정렬, Copperleaf·IFS 등 상용 솔루션 사용
- **성공 사례**: 미국 전기유틸리티 + Black & Veatch, $20억 그리드 현대화 계획 규제 승인
- **현실 문제**: 제한된 예산, 공급망 위기, 인력 부족 속에서 AIP 실행률 저하 지속

### 4.2 현장 GAP 발생 메커니즘

```
[본사 계획] CNAIM Risk 정렬 → 투자 우선순위 리스트
         ↓
[현장 실행] 예산 부족 + 정전 불가 시간대 + 인력 없음 + 현장 접근 불가
         ↓
[결과] 계획 집행률 저하, 임시방편 대응, 실제 위험 자산 방치
```

**GAP의 근본 원인 (문헌 분석):**
1. 화폐화 Risk만으로 선정 → 현장 실행가능성 완전 무시
2. 동일 Risk 자산 수백 개 → 어느 것부터 할지 현장 결정 기준 없음
3. 전문가 판단 (현장 경험) 의사결정 프로세스에서 배제

## 7. 본 연구 방법론 설계 최종 시사점

### 7.1 확정 방법론 체계

```
[데이터 준비]
  CNAIM v2.1 R 패키지 → 합성 자산 1000기 생성
  (변압기·개폐기·선로 → PoF, CoF, Monetised Risk)

[Stage 1: 5종 AHP 가중치 비교]
  ① Classical AHP  → w₁ (기준선)
  ② Fuzzy AHP      → w₂ (TFN, 기하평균법)
  ③ Delphi-AHP     → w₃ (3라운드, CVR≥0.444)
  ④ ANP            → w₄ (슈퍼매트릭스, SuperDecisions)
  ⑤ AI-AHP         → w₅ (ML 가중치 학습 또는 일관성 보정)
  → 자산별 종합점수 S_j^(k) (k=1..5)

[Stage 2: 최적화]
  0-1 ILP: max Σ S_j·x_j
           s.t. 예산, 정전시간, 인력, 의무교체
  솔버: Python PuLP / Pyomo

[검증]
  Baseline(CNAIM-only) vs 5종 AHP 결과 비교
  Spearman 순위상관, Top-N 변동, 예산활용률
  민감도: 가중치 ±20%, 예산 ±30%
```
